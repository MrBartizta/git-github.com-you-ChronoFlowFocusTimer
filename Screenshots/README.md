# App Store Screenshots

Recommended targets (iPhone):
- 6.1" (iPhone 15 Pro)
- 6.7" (iPhone 15 Pro Max)

Steps:
1. Run the app on the desired simulator/device.
2. Navigate to each screen:
   - Hero Timer
   - Preset Selection (open Presets)
   - Analytics
   - Settings
3. Hide debug UI and ensure time shows a clean value (e.g., 25:00).
4. Capture screenshots using Xcode or the simulator.

Notes:
- Use portrait only.
- Ensure the globe glow is visible near countdown end.
