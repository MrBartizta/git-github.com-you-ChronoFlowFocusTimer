# ChronoFlow App Store Checklist

Build & QA
- Build and run on device (iPhone 15/15 Pro recommended)
- Timer counts down to 00:00 and switches to break
- Ambient sound ramps up toward zero and mutes correctly
- Custom time applies and resets timer
- Presets selection updates timer
- No crash on background/foreground

Assets
- App icon set complete
- 6.1" and 6.7" screenshots captured

Metadata (App Store Connect)
- App name: ChronoFlow
- Subtitle
- Description
- Keywords
- Privacy policy URL
- Support URL
- Marketing URL (optional)

Compliance
- No unused permissions (HealthKit removed)
- No background modes unless required
